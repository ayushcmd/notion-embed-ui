What is an Array?

An Array is a contiguous collection of elements of the same data type stored in consecutive memory locations. It provides constant-time access to elements using zero-based indexing.

Key Characteristics:

  • Contiguous memory allocation
  • Fixed size (static arrays)
  • Same data type for all elements
  • Zero-based indexing
  • O(1) random access
  • Cache-friendly performance

Intermediate Operations

üîÑ Reverse Array

Two-pointer: swap from both ends

void reverse(int arr[], int n) { int left = 0, right = n-1; while (left < right) { swap(arr[left++], arr[right--]); } } // O(n)

üîÄ Rotate Array

Reversal algorithm

void rotate(int arr[], int n, int k) { k = k % n; reverse(arr, 0, n-1); reverse(arr, 0, k-1); reverse(arr, k, n-1); }

‚ûï Prefix Sum

Preprocess for O(1) range queries

void buildPrefix(int arr[], int prefix[], int n) { prefix[0] = arr[0]; for (int i = 1; i < n; i++) prefix[i] = prefix[i-1] + arr[i]; } int rangeSum(int prefix[], int l, int r) { return (l == 0) ? prefix[r] : prefix[r] - prefix[l-1]; }

üî¢ Kadane's Algorithm

Maximum subarray sum

int maxSubarraySum(int arr[], int n) { int maxSoFar = arr[0], maxEndingHere = arr[0]; for (int i = 1; i < n; i++) { maxEndingHere = max(arr[i], maxEndingHere + arr[i]); maxSoFar = max(maxSoFar, maxEndingHere); } return maxSoFar; }

üìê Sliding Window

Max sum of subarray size k

int maxSumK(int arr[], int n, int k) { int windowSum = 0, maxSum = 0; for (int i = 0; i < k; i++) windowSum += arr[i]; maxSum = windowSum; for (int i = k; i < n; i++) { windowSum += arr[i] - arr[i-k]; maxSum = max(maxSum, windowSum); } return maxSum; }
üí° QUICK TIP: Two pointers = O(n), beats O(n¬≤) brute force
üí° QUICK TIP: Sliding window = subarray optimization

Common Techniques & Patterns

üéØ Two-Pointer Technique

When: Sorting, palindrome, pair finding

Variants:

  • Opposite direction (reverse, palindrome)
  • Same direction (remove duplicates)
  • Sliding window (subarray problems)
// Two Sum sorted array int twoSum(int arr[], int n, int target) { int left = 0, right = n-1; while (left < right) { int sum = arr[left] + arr[right]; if (sum == target) return true; else if (sum < target) left++; else right--; } return false; }

üîç Binary Search Variations

When: Sorted arrays, search space reduction

// Binary search int binarySearch(int arr[], int n, int target) { int left = 0, right = n-1; while (left <= right) { int mid = left + (right - left)/2; if (arr[mid] == target) return mid; else if (arr[mid] < target) left = mid + 1; else right = mid - 1; } return -1; }

// Count frequencies void countFreq(int arr[], int n) { unordered_map freq; for (int i = 0; i < n; i++) freq[arr[i]]++; }

Summary & Decision Tree

Key Properties Summary

  • Contiguous memory allocation
  • O(1) random access by index
  • Fixed size (static) or dynamic (vector)
  • Excellent cache locality
  • Same data type required
  • Zero-based indexing
  • No pointer overhead
  • Insertion/deletion requires shifting - O(n)
