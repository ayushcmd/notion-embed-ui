What is Recursion?

A Recursive Function is a function that calls itself to solve smaller instances of the same problem. It breaks down complex problems into simpler subproblems until reaching a base case.

Backtracking (Recursion + Choice)

Backtracking = Try all possibilities + Backtrack if wrong path

Pattern: Choose â†’ Explore â†’ Undo

// Generate All Permutations void permute(string s, int l, int r) { if (l == r) { cout << s << endl; return; } for (int i = l; i <= r; i++) { swap(s[l], s[i]); // CHOOSE permute(s, l+1, r); // EXPLORE swap(s[l], s[i]); // UNDO (Backtrack) } } // permute("ABC", 0, 2) â†’ ABC, ACB, BAC, BCA, CAB, CBA

  • BACKTRACK: Undo karo (wapas original state)

  • Position 0 pe 'B' rakho â†’ Recursion â†’ Undo
