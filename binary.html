What is Binary?

A Binary Number System is a base-2 positional numeral system using only two digits: 0 and 1. It's the foundation of all digital computing, where 0 represents OFF and 1 represents ON (electrical states).

Key Terminology:

  • Bit: Binary digit (0 or 1) - smallest unit
  • Byte: 8 bits (256 values: 0-255)
  • Nibble: 4 bits (16 values: 0-15)
  • Word: 32-bit or 64-bit (platform-dependent)
  • MSB: Most Significant Bit (leftmost)
  • LSB: Least Significant Bit (rightmost)
  • Set bit: Bit with value 1
  • Clear bit: Bit with value 0

Bit Manipulation Techniques

1. Check if i-th bit is set

bool isSet(int n, int i) { return (n & (1 << i)) != 0; }

2. Set i-th bit

int setBit(int n, int i) { return n | (1 << i); }

3. Clear i-th bit

int clearBit(int n, int i) { return n & ~(1 << i); }

4. Toggle i-th bit

int toggleBit(int n, int i) { return n ^ (1 << i); }

5. Count set bits (Brian Kernighan's Algorithm)

int countSetBits(int n) { int count = 0; while (n) { n &= (n - 1); // Removes rightmost set bit count++; } return count; } // Built-in function int count = __builtin_popcount(n);

6. Check if power of 2

bool isPowerOf2(int n) { return n > 0 && (n & (n - 1)) == 0; }

Power of 2 has only one set bit: 1000, 0100, 0010, 0001

7. Get rightmost set bit

int rightmostSetBit(int n) { return n & -n; } // Example: 1010 & (-1010) = 0010

8. Swap without temp variable

void swap(int& a, int& b) { a ^= b; b ^= a; a ^= b; } // XOR swap - clever but not always faster
üí° XOR PROPERTIES: a ^ a = 0, a ^ 0 = a, a ^ b ^ a = b. These make XOR perfect for finding unique elements and swapping!

Common Binary Patterns

Pattern 1: Single Number (XOR all)

// Find element appearing once (others twice) int singleNumber(vector<int>& nums) { int result = 0; for (int num : nums) result ^= num; return result; // XOR cancels duplicates }

Pattern 2: Missing Number

int missingNumber(vector<int>& nums, int n) { int xorAll = 0, xorArr = 0; for (int i = 0; i <= n; i++) xorAll ^= i; for (int num : nums) xorArr ^= num; return xorAll ^ xorArr; }

Pattern 3: Subsets Generation

vector<vector<int>> subsets(vector<int>& nums) { int n = nums.size(); vector<vector<int>> result; for (int mask = 0; mask < (1 << n); mask++) { vector<int> subset; for (int i = 0; i < n; i++) { if (mask & (1 << i)) subset.push_back(nums[i]); } result.push_back(subset); } return result; }

Pattern 4: Reverse Bits

uint32_t reverseBits(uint32_t n) { uint32_t result = 0; for (int i = 0; i < 32; i++) { result <<= 1; result |= (n & 1); n >>= 1; } return result; }

Summary & Decision Tree

Key Properties

  • Foundation of all digital computing (base-2)
  • Only two digits: 0 (OFF) and 1 (ON)
  • O(1) bitwise operations - extremely fast
  • Six core operators: &, |, ^, ~, <<, >>
  • Powerful for optimization and low-level programming
  • Essential for flags, masks, state compression
  • XOR properties enable clever algorithms
